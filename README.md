## 🚀 How to Run Any Project

1. Make sure Python is installed on your system
2. Clone this repo:
   ```bash
   git clone https://github.com/your-username/python-projects.git
   ```
3. Navigate to any project folder:
   ```bash
   cd 01-number-guessing-game
   ```
4. Run the file:
   ```bash
   python main.py
   ```
